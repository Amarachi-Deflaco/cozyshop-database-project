# CozyShop Database Project 

##  Project Overview

The **CozyShop Database Project** is a relational database system designed to manage the operations of a small business. It focuses on organizing and managing data related to products, customers, orders, and transactions using SQL.

---

##  Objectives

* Design a structured relational database
* Apply normalization principles
* Create relationships between tables
* Write SQL queries for data manipulation and retrieval

---

##  Tools & Technologies

* MySQL
* MySQL Workbench
* SQL (Structured Query Language)

---

## 🗂️ Database Structure

The database includes the following tables:

* **Customers** – stores customer information
* **Products** – stores product details
* **Orders** – records customer orders
* **Order_Details** – manages products in each order

### Relationships:

* One customer can place many orders
* One order can contain multiple products
* Each product can appear in multiple orders

---

## Features & Queries

This project includes:

* Table creation using `CREATE TABLE`
* Data insertion using `INSERT INTO`
* Data retrieval using `SELECT`
* Filtering using `WHERE`
* Joining tables using `JOIN`
* Aggregations (e.g. `SUM`, `COUNT`)
